## Criando turmas

Uma turma é criada a partir de uma oferta de disciplina. Para criar:

1. Acesse **Turmas** no menu lateral
2. Selecione a oferta correspondente
3. Defina os horários de aula
4. A turma fica disponível para matrícula

## Horários

Os horários são definidos por:

- **Dia da semana**
- **Horário de início e fim**
- **Sala** (opcional)

O sistema verifica automaticamente conflitos de horário para professores.

## Frequência

O professor lança a frequência por aula. O sistema calcula automaticamente o percentual de presença de cada aluno e alerta casos de risco de reprovação por falta.

## Notas

O lançamento de notas é feito pelo professor diretamente na turma. A média final é calculada conforme a configuração da disciplina.
