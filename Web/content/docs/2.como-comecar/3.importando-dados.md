---
title: Importando dados
description: Como migrar cursos, disciplinas e alunos de um sistema anterior.
---

Se você está migrando de outro sistema, o Estud oferece importação em lote via planilha.

## O que pode ser importado

- **Disciplinas** — código, nome, carga horária
- **Cursos** — nome, tipo, grade curricular
- **Alunos** — dados cadastrais e matrícula
- **Professores** — dados cadastrais e vínculo com disciplinas

## Formato da planilha

Cada entidade tem seu próprio modelo de planilha. Baixe os modelos em **Configurações > Importação**.

## Ordem recomendada

Importe nesta ordem para evitar dependências:

1. Disciplinas
2. Cursos (com as disciplinas já criadas)
3. Professores
4. Alunos

::callout{type="warning"}
Verifique os dados antes de importar. Registros duplicados podem gerar inconsistências.
::
