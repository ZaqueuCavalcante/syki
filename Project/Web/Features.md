# Features

## Usuários

- Cadastro, login, social-login, logout
- Esqueci minha senha, redefinir senha

- Editar perfil
- Vincular empresa
- Mudar de conta (impersonar uma empresa)

- Criar usuário
- Vincular empresas ao usuário
- Excluir usuário

## Compras

- Comprar créditos / adicionar saldo
- PIX, cartão de crédito, boleto, débito online

## DocCheck

- Histórico de Verificações
- Gerar link de verificação
- Dashboard

## Consultas

- Consultas Pessoa Física 
    - PF
    - PF + Quod
    - Situação do CPF - Receita Federal

- Consultas Pessoa Jurídica 
    - PJ
    - PJ + Quod
    - Situação do CNPJ - Receita Federal
    - Optantes pelo Simples Nacional - Receita Federal

- Consultas em lote
    - Subir um Excel com os CPFs/CNPJs

- Histórico de consultas
    - Filtros e dados agregados

## Extrato

- Todas as consultas feitas em um período + quanto cada uma custou
