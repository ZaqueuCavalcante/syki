O **Estud** é um sistema de gestão acadêmica (SGA) open-source desenvolvido para simplificar a administração de instituições de ensino.

## Principais conceitos

### Instituição

Cada instituição é um tenant isolado dentro da plataforma. Todos os dados — cursos, alunos, professores — são vinculados à instituição do usuário logado.

### Perfis de acesso

A plataforma possui quatro perfis principais:

| Perfil | Responsabilidades |
|--------|-------------------|
| **Acadêmico** | Gestão de cursos, grades, turmas e períodos |
| **Professor** | Lançamento de notas e frequência |
| **Aluno** | Acompanhamento de matrículas e notas |
| **Administrador** | Configurações da instituição e segurança |

### Multi-tenant

Cada instituição opera de forma completamente isolada. Não há compartilhamento de dados entre instituições diferentes.

## Tecnologia

O Estud é construído com:

- **Backend**: ASP.NET Core + PostgreSQL
- **Frontend**: Nuxt.js + Nuxt UI
- **Infraestrutura**: Docker, Railway, GitHub Actions
